# Security Policy

## Supported Versions

Odinfra is pre-1.0. Security fixes are released for the latest published version.

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories for `ozkozturk/odinfra` when available. If advisories are unavailable, open a minimal issue that does not include exploit details and ask for a private contact path.

Avoid posting credentials, private project files, or runnable exploit details in public issues.
